# 测试报告

## 当前结果

最近一次通过的验证：

```bash
pnpm lint
pnpm test
pnpm build
pnpm test:e2e
```

## 单元测试

当前覆盖：

- 敏感词检测。
- 手机号校验。
- 手机号脱敏。
- Cookie 会话 token 生成、哈希和校验。
- 生产环境认证配置校验：强 `JWT_SECRET`、关闭 `MOCK_SMS`、Cookie Secure。
- AI 输出 schema。
- AI 草稿比分数量校验。
- AI 草稿敏感词检测。
- 预测、复盘、广告合作信息保存前敏感词检测。

## E2E 测试

当前覆盖：

- 首页展示赛事和免责声明。
- 游客浏览比赛列表和详情。
- 复盘页和广告合作页渲染。
- 复盘页、比赛详情页和分享海报免责声明可见。
- 手机号验证码登录、Prisma Session 和退出。
- 管理员访问 Prisma 优先的后台只读页面。
- 普通用户访问后台被 403 拦截。
- 管理员创建/编辑赛事，接口返回成功并写入 `AuditLog`。
- 管理员批量导入赛事，接口返回成功并写入 `AuditLog`。
- 管理员创建/编辑预测，接口返回成功并写入 `AuditLog`。
- 管理员创建/编辑复盘，接口返回成功并写入 `AuditLog`。
- 管理员修改用户角色/状态，接口返回成功并写入 `AuditLog`。
- 管理员创建/编辑广告计划和素材，接口返回成功并写入 `AuditLog`。
- 管理员保存模型/Prompt 设置和赛事数据源设置，接口返回成功并写入 `AuditLog`。
- 广告主访问 Prisma 优先的投放后台。
- 普通用户访问广告主后台被 403 拦截。
- 广告 active/events/leads API，events 写入 `AdEvent`，leads 写入 `LeadForm`。
- 分析师调用 AI 草稿 API。
- 分析师可将 AI 草稿保存为预测草稿。
- 普通用户调用 AI 草稿 API 被 403 拦截。
- 管理员发布/下架预测，接口返回成功并写入 `AuditLog`。
- 管理员发布/下架复盘，接口返回成功并写入 `AuditLog`。
- 管理员审核广告，接口返回成功并写入 `AuditLog`。
- 首页生成分享海报。
- 公共 H5 页面、个人中心、广告主后台和管理后台只读列表在 Prisma seed 数据下可浏览。
- 登录用户可在比赛详情页收藏比赛、设置开赛提醒，并在个人中心看到刷新后的数据。
- 预测/复盘保存、广告线索提交命中敏感词时返回 422。
